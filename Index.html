Google Sheets
Cole a URL do seu Google Apps Script (Web App) para sincronização automática dos registros. Os dados são enviados via POST e você controla tudo na sua conta Google.
📋 Como configurar (5 minutos):
1. Abra Google Sheets → Extensões → Apps Script
2. Cole o código do apps-script.gs (incluído abaixo)
3. Implante como Web App com acesso "Qualquer pessoa"
4. Copie a URL gerada e cole acima
Apps Script — código pronto
Cole este código no Google Apps Script da sua planilha.
function doPost(e) {
  var sheet = SpreadsheetApp
    .getActiveSpreadsheet()
    .getSheetByName("Registros");
  if (!sheet) {
    sheet = SpreadsheetApp
      .getActiveSpreadsheet()
      .insertSheet("Registros");
    sheet.appendRow([
      "ID","Colaborador","Tipo",
      "Horário","Data","Sync"
    ]);
  }
  var data = JSON.parse(e.postData.contents);
  data.forEach(function(r) {
    sheet.appendRow([
      r.id, r.nome, r.tipo,
      r.horario, r.data,
      new Date().toISOString()
    ]);
  });
  return ContentService
    .createTextOutput(
      JSON.stringify({ok:true})
    )
    .setMimeType(
      ContentService.MimeType.JSON
    );
}
